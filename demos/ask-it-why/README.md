# Demo: ask it why

Story: [kyzo#38](https://github.com/kyzodb/kyzo/issues/38) · Epic: [kyzo#41](https://github.com/kyzodb/kyzo/issues/41)

Live loop, sixty seconds: a recursive query returns an answer; the answer unfolds its verified proof
tree; one ground fact is retracted; the belief revises; an as-of query shows what was believed
before. Retrieval, provenance, and time travel in one demonstration.

Status: not yet built. Needs engine provenance and time travel end-to-end
([kyzo#4](https://github.com/kyzodb/kyzo/issues/4)).
